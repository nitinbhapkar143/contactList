# README #

* Quick summary
  This is a simple contact list App built using MEAN stack.In this app, you can add contact with the following fields :
 1. Name
 2. Email
 3. Mobile Number
You can add, update or delete the contacts or you have option to delete all the contacts. The app is built using REST api's.

* Version 1.0.0

* Summary of set up
  Make sure you have Node and MongoDB installed. Just clone this directory and Type npm install. This will install all the dependencies which this project have And then type node server. Now open browser and goto http://127.0.0.1:3001
